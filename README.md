# Stellaris map coverter from raw stellar data

1. To convert raw `data.txt` to config file `data.json` put `data.txt` to `convert` folder and run
```
yarn convert
```
or
```
npm run convert
```

2. To create map for Stellaris from `data.json` run
```
yarn create-map
```
or
```
npm run create-map
```
